# Fonyód

Fonyód város Somogy vármegyében, a Fonyódi járás központja. Népszerű üdülőhely és kikötőváros a Balaton déli partján. Hozzá tartozik Alsóbélatelep, Bélatelep és Fonyódliget is. Lakossága 5201 fő.
